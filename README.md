# Customer Review Sentiment Analysis

## Overview
This project analyzes large-scale customer reviews using natural language processing techniques to understand sentiment trends and key complaint themes.

## Dataset
568,000 customer reviews dataset.

## Methods
- Sentiment labeling
- NLP preprocessing
- Word frequency analysis
- Complaint keyword extraction
- Sentiment visualization
- Transformer-based sentiment inference

## Key Insights
- Majority of reviews show positive sentiment.
- Negative feedback frequently mentions taste, packaging, and product quality.
- NLP techniques reveal recurring complaint themes.

## Tools Used
Python, Pandas, NLTK, Transformers, Seaborn
